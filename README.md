# Environment Variables

Create a .env file in the root of your project and add:
**VITE_GEMINI_API_KEY=**
**VITE_GEMINI_URL=**

# Running the Project

- Install dependencies:
  **npm install**


- Start development server:
 **npm run dev**


# ⚠️ API Rate Limits
  -Maximum ~20 requests in a short time window


# 📌 Notes
 - This project uses the gemini-3-flash-preview model
